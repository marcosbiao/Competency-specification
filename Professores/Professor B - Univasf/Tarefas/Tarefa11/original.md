Faça um programa para calcular o cosseno de x, para x variando de 0 até 6,3 (inclusive), de 0,1 em 0,1. O programa deverá imprimir x e o valor correspondente do cosseno. O cálculo do cosseno deverá ser implementado através de um subprograma que utilize a série:
``` cos(x) = 1 - x²/2! + x⁴/4! - x⁶/6! + x⁸/8! + ... ```
Considerar as primeiras 30 parcelas da série para a obtenção do cosseno de x.
