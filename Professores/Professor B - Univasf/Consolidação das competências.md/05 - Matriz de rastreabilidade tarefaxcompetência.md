# Matriz de rastreabilidade tarefa x competência consolidada

## 1. Escopo

Esta matriz relaciona cada tarefa do projeto às competências consolidadas obtidas após a unificação agressiva das 49 competências originais.
A marcação foi feita a partir do mapeamento entre competências originais e consolidadas definido para este projeto.

Critério de marcação: uma competência consolidada foi associada à tarefa quando ao menos uma competência original daquela tarefa foi mapeada para essa competência consolidada.

| Código | Competência consolidada |
|---|---|
| C1 | Traduzir problemas em algoritmos e implementar programas imperativos introdutórios com fluxo verificável |
| C2 | Manipular dados com tipos primitivos, constantes, variáveis e expressões, preservando a coerência do estado do programa |

| C4 | Controlar o fluxo com decisões, formulando condições lógicas corretas |

| C7 | Representar, percorrer e transformar estruturas indexadas com vetores e matrizes, preservando rastreabilidade posicional |
| C8 | Processar estruturas indexadas de forma iterativa, com percorrimento, atualização e parada segura |
| C9 | Modularizar programas em funções ou subprogramas coesos, com interfaces adequadas e integração consistente ao fluxo principal |

| C11 | Construir subprogramas recursivos com progressão verificável |
| C12 | Desenvolver e justificar sistemas simples com persistência e consulta em arquivos binários |

## 6. Leitura analítica

A competência com maior reuso é **C6**, cada uma associada a **6 tarefas**.
As competências **C1**, **C2**, **C3**, **C4** e **C9** aparecem em **5 tarefas** cada.
As competências **C7** e **C8** aparecem em **4 tarefas** cada.
A competência **C5** aparece em **2 tarefas**.
As competências **C10**, **C11**, **C12** aparecem em apenas **1 tarefa** cada, indicando caráter mais específico no conjunto atual.

Em termos de cobertura, a tarefa com maior diversidade de competências consolidadas é **Tarefa09**, com **4 competências consolidadas associadas**.
As tarefas com menor diversidade de competências consolidadas são **Tarefa01, Tarefa07, Tarefa11, Tarefa15, Tarefa16**, cada uma com **2 competências consolidadas associadas**.

Esse resultado pode ser usado como base para análise de reuso, balanceamento curricular e planejamento de recomendação de recursos educacionais por competência consolidada.
