
Uma companhia resolveu cadastrar todos os funcionários em um sistema computacional.

Você foi contratado para desenvolver o sistema em questão. Será necessário armazenar os seguintes dados do funcionário:

| Campo | Descrição |
|---|---|
| `codigo` | Código dos funcionários |
| `nome` | Nome do funcionário (até 40 caracteres) |
| `cargo` | Cargo do funcionário (até 20 caracteres) |
| `salario` | Valor do salário do funcionário |
| `telefone` | Telefone de contato do funcionário |
| `status` | `"+"` se ativo; `"*"` se promovido a cliente |

O sistema deverá ter as opções para:

a) Inclusão de funcionários

b) Consulta de funcionários pelo código ou nome

c) Alteração nos dados do funcionário

d) Exclusão de funcionário

e) Relatório de funcionários
