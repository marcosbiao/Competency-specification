A clínica veterinária AMOMEUPET precisa de um sistema
para calcular o salário dos veterinários, com base na lista
de serviços que eles realizam ao longo do período de um
mês.
- Cada veterinário realiza mensalmente até 20 serviços.
- Cada serviço precisa ter o código (que é uma letra) que o
identifica, o código numérico do paciente, e o valor cobrado pela
clínica.
- A lista de códigos de serviço é: ‘C’ – Consulta; ‘E’ – Exame; ‘U’ –
Atendimento de urgência.
- Cada tipo de serviço possui uma comissão associada, de modo que
o valor pago pelo cliente não é o mesmo que o veterinário recebe.
O veterinário recebe a comissão.
- Os valores de comissões:
    - Consulta: 80%;
    - Exame: 50%;
    - Atendimento de urgência: 40%.
- Ao final da inserção dos serviços realizados o programa exibe um
relatório com as informações de todos os serviços prestados e o
valor final, que é o salário do veterinário.


Exemplo de relatório:

RELATORIO MENSAL

Nome do veterinario: Marcio

Quantidade de atendimentos realizados: 3

Procedimento/cod paciente/valor/valorfinal

SALARIO DO PROFISSIONAL: R$ 220.00
